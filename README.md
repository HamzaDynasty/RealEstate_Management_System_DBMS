# RealEstate_Management_System_DBMS
Real Estate Management System (REMS) REMS is a comprehensive web app for real estate operations.   Tech Stack: Built on Node.js/Express.js and MySQL.    Access Control: Uses RBAC for three roles:   Admin/Agency    Agent    Client    Functionality: Manages the full Property Lifecycle (listing,transaction,payment). Database: Highly normalized to 3NF.
